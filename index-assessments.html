Assessments

Learn by Playing

Assess your knowledge that you have learned from the learn section of the app and put it to the test by playing different types of games.

Play different kinds of activities in order to assess your knowledge of the periodic table.
  • Identification
  • Multiple Choice
  • Guess the element
  • Elements to Compounds
  • Drop the element
  • Complete the table

How does Identification work?

  • Uploading of Test

    The tests within the game are created by the teacher. The teacher generates a set of tests, uploads them to the game, and assigns a unique quiz ID to each test. Students can access the tests by using the corresponding quiz ID.

    In order to create a quiz, click the button below.

  • Take the test

    Now that a quiz has been successfully made, the students can now take on our Elementum App by entering the code.

  • See the results

    See the result of the test and see where you were wrong
