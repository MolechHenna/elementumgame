Learn - Periodic Table

Learn the individual elements

Each individual element has its own unique property that you can read about in this part of the app. Learn and master each element and be the best among your peers.

A fun way to learn about each element of the periodic table.

How does it work?

  • Choose an Element

    By tapping on a specific element you will be able to choose between information, play, and leaderboard. The way it works is that the information works as a way for you to study about the Elements and the play option is for you to be able to play a minigame about that element.

  • Play the minigame to unlock more Elements

    If you click play, you will be going through a set of minigame in which if you manage to finish it you will be able unlock more Elements for you to learn from.

  • Check the leaderboard and compete with friends

    After completing a minigame for the Element, you can see the leaderboard for each minigame and see who did the minigame the fastest so long as they get a high score.
